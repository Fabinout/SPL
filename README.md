# SPL вАФ Structured Parenthesized Language

A **low-level, stack-based VM language** designed to be easily generated by LLMs and trivially parsed. This repository contains the language specification, Python implementation, and several example applications.

## рЯЪА Quick Start

### What is SPL?
SPL is a minimal, parenthesized assembly-like language for a stack-based virtual machine. It's designed to be:
- **Easy to generate** вАФ LLMs naturally produce S-expressions
- **Simple to parse** вАФ Recursive descent or stack parser
- **Compact & deterministic** вАФ Suitable for embedded/resource-constrained systems
- **Game-friendly** вАФ Includes I/O for keyboard, video, audio, mouse

### Example Program
```lisp
; Print "Hello, World!" using string output
(push 72) (store 0x0400)   ; 'H'
(push 101) (store 0x0401) ; 'e'
; ... (store rest)
(push 0) (store 0x0408)    ; null terminator
(print-cstring 0x0400)     ; output complete string
(halt)
```
